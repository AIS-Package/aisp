# aisp.base

Base Classes and Core Utilities.

> **Module:** `aisp.base`

## Overview

This module provides the fundamental classes and utilities that serve as the basis for all
Artificial Immune Systems algorithms implemented in the AISP package.

## Classes

| Class                                    | Description                                        |
|------------------------------------------|----------------------------------------------------|
| [`BaseClassifier`](./base-classifier.md) | Abstract base class for classification algorithms. |
| [`BaseClusterer`](./base-clusterer.md)   | Abstract base class for clustering algorithms.     |
| [`BaseOptimizer`](./base-optimizer.md)   | Abstract base class for optimization algorithms.   |
