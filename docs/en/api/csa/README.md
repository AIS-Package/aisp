# aisp.csa

Module (CSA) Clonal Selection Algorithm.

> **Module:** `aisp.csa`

## Overview

CSAs are inspired by the process of antibody proliferation upon detecting an antigen, during which
the generated antibodies undergo mutations in an attempt to enhance pathogen recognition.

## Classes

| Class                     | Description                                                                                                                                                                          |
|---------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [`AIRS`](./airs.md)       | A supervised learning algorithm for classification tasks based on the clonal selection principle.                                                                                    |
| [`Clonalg`](./clonalg.md) | Implementation of the clonal selection algorithm for optimization, adapted for both minimization and maximization of cost functions in binary, continuous, and permutation problems. |
