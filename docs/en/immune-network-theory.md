# Immune network theory

This technique was introduced by **Niels Jerne (1974)** and models the immune system as a dynamic network, in which
cells and molecules are capable of recognizing each other. [^1]

---

The Artificial Immune Network can be applied in different contexts, such as:
- **Clustering**
- **Optimization**
- **Classification**

## Package implementation

### Rede Imunológica Artificial para Agrupamento e Compressão ([AiNet](./api/ina/ai-net.md))

AiNet is a clustering algorithm based on immune network theory that uses clonal selection and affinity maturation
to compress data and identify groups [^2].
